Forked from: http://github.com/bluerhinos/phpMQTT originally by Andrew Milsted

A simple php class to connect/publish/subscribe to an MQTT broker

To use
-----------------------
Unlike other PHP MQTT classes and libraries you don't need to install anything other than using a copy of the phpMQTT.php file

Other classes tested would have the connection closed if there were too many bytes sent, but this class doesn't have the same limitation.
